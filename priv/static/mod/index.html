Important Info

Admins are people who happen to play on the server often and would like it to remain populated. Being an admin allows them to #Save the server when something is wrong.
If there's a hacker or the nextmap is fairly dangerous, you can intervene to help the server remain active.

Note: substrings work, for example "!ban koga 0 scary" would ban a player named Kogasa. The 0 means permanent.
