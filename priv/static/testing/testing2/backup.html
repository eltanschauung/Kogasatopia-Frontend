April 13th, Koggy Admin
Happy Passover! The new Kogasatopia website just dropped; it's like Teamfortress.com but more Kogasatopia oriented... we even have a blog now. Everything related to the server will now be featured here.
The last few Kogasatopia seasonal updates have been focused on weapon changes, but this time we have few; that's because we'd run out if we blew our load right now. Instead, along with some new features like being able to nominate workshop maps by name, we've added the Versus Saxton Hale gamemode to the server for the first time, which became possible with Vscript on March 17th.

New mods page just dropped

Babe, what if there were Kogasatopia .vpk files that would make the TF2 world subtly more Touhou related? They actually exist; we just haven't had a place to list all of them until now. Each one has been carefully selected by how faithfully it fits TF2's artstyle since Kogasatopia occasionally cares about quality control.
Pictured is our important new mod that replaces most of TF2's signs and decals with Kogasatopia related ones, and it changes the ambulance and Dispenser textures slightly too. Other important vpk files include a Marisa hat that replaces the Ghostly Gibus and sexy Touhou calendars, which are soulful because they match the original poses of the TF2 calendars.
